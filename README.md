# Ohjelmistotekniikka, harjoitustyö

Sovelluksen tarkoituksena on tarjota käyttäjälle työkalu *tehtävien* hallintaan, joka keskittyy **viikkotason** suunnitteluun perinteisen päivätason aikataulutuksen sijaan.
